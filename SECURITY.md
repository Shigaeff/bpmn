# Política de Segurança

## Reportar uma vulnerabilidade

Se você encontrar uma vulnerabilidade de segurança, **NÃO abra uma issue pública**.

Envie um email para **vladimir@torgalimentos.com** com:

- Descrição da vulnerabilidade
- Passos para reproduzir
- Impacto potencial

Você também pode usar o recurso de **Private Vulnerability Reporting** do GitHub neste repositório.

Responsável: Vladimir Shigaeff
